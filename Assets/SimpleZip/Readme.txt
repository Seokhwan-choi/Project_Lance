Simple Zip is a crossplatform plugin to compress and decompress data using Zip.

Features:
- Compress string and binary data
- Compress files and directories to ZIP-archive
- Any platform: iOS, Android, Desktop and WebGL

Usage:
	Zip.Compress(PLAIN_TEXT)
	Zip.CompressToString(PLAIN_TEXT)
	Zip.Decompress(COMPRESSED_TEXT)
	Zip.Decompress(BYTE_ARRAY)
	Zip.CompressDirectory(DIRECTORY_PATH, ARCHIVE_PATH)
	Zip.CompressFile(FILE_PATH, ARCHIVE_PATH)
	Zip.DecompressArchive(ARCHIVE_PATH, DIRECTORY_PATH)
